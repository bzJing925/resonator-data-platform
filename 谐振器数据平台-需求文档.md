# 谐振器/滤波器测试数据平台 — 需求文档

**版本**：v0.2
**日期**：2026-05-09
**目的**：替代 Excel，搭建支持上传、入库、可视化分析的网页平台

---

## 1. 背景与目标

- 当前测试数据用 Excel 管理，几十万条数据已严重卡顿
- 数据格式标准（S2P / S1P + 提取参数表），适合统一入库
- 目标：**多人浏览器访问**，支持上传、查询、交互式作图、导出
- **优先级**：先做**谐振器**系统；滤波器系统结构类似，二期实现

---

## 2. 数据情况与处理流程

### 2.1 数据规模

| 项 | 说明 |
|---|---|
| 总量 | 十几 GB；约 18 个批次，每批 2–3 个 wafer，每 wafer ~10000个器件 |
| 单器件原始文件 | ~600 KB |
| 总记录数 | 几十万条（每条对应一个器件） |
| 变量类型 | 类别型（type）、几何型（X/Y 坐标对）、数值型（Q、频率、S 参数等） |

### 2.2 数据流转链路

```
用户压缩包/文件夹（=一个批次）
  ├── 包含若干 S2P 文件（DUT、OPEN、SHORT 校准件）
  │
  ▼  [步骤 1：拆分 S2P → S1P]   （仅谐振器路径需要；滤波器跳过此步）
  │
  S1P 文件集合（含 DUT、OPEN-n、SHORT-n）
  │
  ▼  [步骤 2：参数提取 + 对照表翻译]
  │     - 用对照表把代号（如 a1-1）翻译成实际参数
  │     - 提取 fs、fp、Zs、Zp、Q 系列、k2eff 等数值
  │
  Excel 参数表（一行 = 一个器件）
  │
  ▼  [步骤 3：可选 De-embedding]   （ShortOpen 校准；用户上传时勾选）
  │
  De-embedded S1P 文件
  │
  ▼  [步骤 4：入库]
  │     - 原始 S2P/S1P  → 文件存储区
  │     - Excel 参数表  → SQL 数据库
  │     - 批次号        = 上传文件夹名
```

### 2.3 变量分类（基于实际表头）

| 类别 | 字段举例 |
|---|---|
| 类别型 | 文件名 / 批次号 / 代号（a1-1）/ ny / fs 等 type 字段 |
| 几何型 | X、Y 坐标（成对，用于版图分布图） |
| 数值型（关键分析列）| `fs(GHz)`、`fp(GHz)`、`Zs(Ω)`、`Zp(Ω)`、`Qs`、`Qp`、`Qs_BodeQ`、`Qp_BodeQ`、`dbqs`、`dbqp`、`BodeQ_fitted`、`BodeQ_smooth`、`Fbode(GHz)`、`k2eff(%)` |

## 3. 功能模块

### 3.1 上传模块

**入口**
- 谐振器（先做）
- 滤波器（占位，二期实现）

**上传形式**
- 上传文件夹或 zip 压缩包
- **文件夹名自动作为批次号**（不需用户手动输入）
- 一次上传 = 一个批次
- **历史数据复用同一入口**，无需单独的导入流程

**对照表**
- 单独上传，与数据文件分离
- 每次上传时**选用对应的对照表**（不同批次/版图可能不同）
- 用于把代号（如 `a1-1`）翻译成实际参数

**可配置项**
1. **频率范围**（如只取 14–16 GHz；默认全频段）
2. **是否执行 De-embedding**（ShortOpen 校准；勾选后调用 `de.py` 流程）
3. 其他脚本选项一律写死/隐藏

**数据清洗规则**
- 仅检查以下数值列：`fs(GHz)`, `fp(GHz)`, `Zs(Ω)`, `Zp(Ω)`, `Qs`, `Qp`, `Qs_BodeQ`, `Qp_BodeQ`, `dbqs`, `dbqp`, `BodeQ_fitted`, `BodeQ_smooth`, `Fbode(GHz)`, `k2eff(%)`
- 任一列为 NA 或空 → 整行丢弃
- 其他列即使为空也保留

### 3.2 存储模块

| 数据类型 | 存储方式 |
|---|---|
| 原始 S2P / S1P 文件 | 文件存储区（保留，供曲线图现读现画） |
| De-embedded S1P 文件 | 文件存储区（如有执行） |
| 提取参数表 | SQL 数据库（供筛选、聚合、散点/箱型图查询） |
| 对照表 | 与批次关联存储 |

### 3.3 可视化模块

**两类数据源**

| 图类型 | 数据来源 | 说明 |
|---|---|---|
| 单器件 S 参数曲线 | 原始 S2P/S1P | 频率 vs S21/dB，现读现画 |
| 散点 / 箱型 / 折线 / 版图分布 | SQL 参数表 | 跨批次、跨器件聚合分析 |

**作图能力**
- X 轴、Y 轴、颜色字段：用户从所有变量中任选（拖拽式最理想）
- 支持图类型：
  - **散点图**（最常用）
  - **箱型图**（看分布）
  - **折线图 / 批次趋势图**
  - **版图分布图**（X/Y 锁定为几何坐标，颜色编码任意参数）
  - **单器件 S 参数曲线**
- 交互式：缩放、hover 显示数值

**筛选**
- 按任意变量值过滤（如"只看 4 GHz 的点"）
- 大数据量自动降采样/聚合，避免卡顿

### 3.4 导出模块

- 图片下载（建议同时支持交互式 HTML 导出）
- 数据导出（CSV / Excel）

### 3.5 部署与访问

- **部署位置**：所内服务器
- **访问方式**：浏览器，无需安装客户端，多用户同时使用
- **权限**：暂不做权限分级（所有用户均可上传 / 查看 / 导出）
